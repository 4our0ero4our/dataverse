# Walrus Wallet Integration

This directory contains Walrus wallet integration and utilities.

## Purpose

- Walrus wallet connection
- Wallet event handlers
- Transaction signing with Walrus
- Account management

## Usage

```typescript
import { connectWalrus } from '@/app/lib/walrus/connector'
import { signTransaction } from '@/app/lib/walrus/signer'
```

## Installation

Install Walrus wallet SDK:
```bash
npm install @mysten/wallet-kit
```
