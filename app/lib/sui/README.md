# Sui SDK Integration

This directory contains Sui SDK integration and utilities.

## Purpose

- Sui client initialization and configuration
- Sui transaction utilities
- Sui account management
- Sui contract interactions

## Usage

```typescript
import { suiClient } from '@/app/lib/sui/client'
import { sendTransaction } from '@/app/lib/sui/transactions'
```

## Installation

Install the Sui SDK:
```bash
npm install @mysten/sui.js
```
